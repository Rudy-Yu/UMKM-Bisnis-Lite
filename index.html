Bantuan & FAQ

Apa itu aplikasi UMKM Bisnis Lite?

UMKM Bisnis Lite adalah aplikasi pencatatan keuangan, stok, pelanggan, hutang/piutang, dan analitik bisnis untuk UMKM. Mudah digunakan, ringan, dan semua data disimpan di perangkat Anda (LocalStorage).
Solusi Mudah Manajemen Bisnis UMKM

Pertanyaan Umum

  • Bagaimana cara backup data?
    Gunakan menu Pengaturan > Backup & Restore Data untuk mengunduh atau memulihkan data.
  • Lupa password login?
    Password default adalah admin123. Jika sudah diganti dan lupa, reset data aplikasi (akan menghapus semua data).
  • Bagaimana cara export/import laporan?
    Gunakan tombol Export CSV dan Import CSV di halaman Laporan.
  • Apakah data saya aman?
    Data hanya tersimpan di perangkat/browser Anda. Untuk keamanan lebih, lakukan backup rutin.
